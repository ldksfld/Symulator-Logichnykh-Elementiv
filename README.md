# СЛЕСЧ — Симулятор логічних елементів та систем числення

Навчальний застосунок для вивчення цифрової логіки та систем числення. Працює як повноцінна програма на Windows — без браузера, без інтернету.

---

## Можливості

**Системи числення**
- Переведення чисел між DEC, BIN, HEX (включно з дробовими)
- Покрокове пояснення кожного перетворення з алгоритмом
- Таблиця відповідності, степені 16, поняття розрядності

**Логічні елементи**
- Drag-and-drop схемний редактор
- Елементи: буфер, НЕ, І, АБО, І-НЕ, АБО-НЕ, XOR, константа, вхід, вивід
- Автоматична симуляція в реальному часі, кольорове відображення сигналів
- Генерація таблиці істинності, ДДНФ, ДКНФ, логічного виразу
- Готові приклади: напівсуматор, повний суматор, OR через NAND та інші
- Збереження та завантаження схем у файл

**Булева алгебра**
- Парсинг і обчислення виразів (`A & B | !C`, `A AND (B XOR C)` тощо)
- Таблиця істинності, ДДНФ, ДКНФ, спрощений вираз із поясненням кроків
- Карта Карно (2–4 змінні) з виділенням груп мінімізації
- Перевірка тотожності двох виразів

**Довідник**
- Таблиця числових систем 0–15
- Закони Булевої алгебри
- Таблиці істинності всіх логічних елементів з формулами

## Встановлення та запуск

### Варіант 1 — Готовий інсталятор (рекомендовано)

1. Завантажити файл `СЛЕСЧ Setup 1.0.0.exe` зі сторінки [Releases](../../releases) на GitHub
2. Запустити, натиснути «Далі» кілька разів
3. На робочому столі з'явиться ярлык — запускати як звичайну програму

> Нічого додатково встановлювати не потрібно.

### Варіант 2 — Запуск з вихідного коду

Потрібен [Node.js 20+](https://nodejs.org) та пакетний менеджер `npm`.

```bash
# 1. Клонувати або розпакувати проєкт
# 2. Перейти до папки проєкту і виконати:

npm install
npm run dev
```

Відкриється вікно застосунку.

### Варіант 3 — Зібрати інсталятор самостійно

```bash
npm install
npm run build
```

Готовий `.exe`-інсталятор з'явиться у папці `release/`.
